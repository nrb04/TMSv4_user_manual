
\section{Website Operate and Manage Classic View}
This section provides an overview of the website in the classic view mode. The classic view provides a user-friendly interface for navigating and managing various aspects of the website.

\subsection{Overview of Classic View}
The classic view of the website consists of several sections, each represented by specific UI elements. Below are the key sections and their functionality:

\begin{itemize}
  \item \textbf{Menu Bar (Pointer 1)}: This is the main navigation panel on the left-hand side of the website. It allows users to access various sections like Dashboard, Traffic, Revenues, QoS, Tariffs Catalog, and others.
  \item \textbf{Classic and Modern View Switch Button (Pointer 2)}: This button allows the user to switch between the classic and modern view of the website. The user can toggle between these views to get a different perspective or user interface.
  \item \textbf{Search on this Page (Pointer 3)}: The search bar enables users to quickly search for specific data or reports on the page.
  \item \textbf{Profile of User (Pointer 4)}: This section displays the logged-in user's profile. It shows their name, role (Administrator or other), and other personal details. Users can access settings and log out from this section.
  \item \textbf{Custom Search (Pointer 5)}: This allows users to apply specific filters or search criteria. It includes options like time range, operators, network categories, and direction for traffic data. Users can enter custom search queries to view the relevant data.
  \item \textbf{User Custom Search Results (Pointer 6)}: Once the filters are applied, the results are displayed here. The data is shown in a graphical format (bar charts, etc.) depending on the selected filters.
  \item \textbf{Administration Section (Pointer 7)}: This section is only visible to users with administrative privileges. Non-admin users will not have access to this section.
\end{itemize}

\subsection{Classic View Description}
The classic view offers a detailed and comprehensive presentation of traffic data. Key elements in this view include:
\begin{itemize}
  \item \textbf{Traffic Data Visualization}: The website displays traffic volume data in a graphical format (e.g., bar charts), with different traffic types and filters applied.
  \item \textbf{Traffic Customization}: Users can filter traffic data by selecting different options such as period, network category, direction, and more.
  \item \textbf{Export Options}: Users have options to export the displayed data in various formats (CSV, Excel, PDF, etc.) for further analysis or reporting.
  \item \textbf{Live Connection Status}: The live connection status (e.g., "Traffic API connected") is shown, which indicates whether the system is receiving real-time traffic data.
\end{itemize}
\textbf{Note:} This section may change once the modern view is shared, but for now, the classic view offers a simple, intuitive interface for monitoring and managing telecom traffic.

\subsection{Website Operate and Manage Modern View}
This section provides an overview of the website in the modern view mode. The modern view offers enhanced features and flexibility, allowing users to interact with data in multiple ways and view traffic information efficiently.

\subsection{Overview of Modern View}
The modern view of the website provides several sections to navigate through traffic data and customize views. Below are the key sections and their functionality:

\begin{itemize}
  \item \textbf{Menu Bar \& Custom Search (Pointer 1)}: This is the main navigation panel on the left-hand side, which allows users to select different categories such as traffic data, periods, operators, network categories, and time ranges for custom searches.
  \item \textbf{User Custom Search Results (Pointer 2)}: After applying the custom search filters, the results are displayed in various formats such as \textit{Graph View}, \textit{Table View}, and \textit{Statistics}. Users can switch between these views to analyze data according to their preferences.
  \item \textbf{Download Options \& Export (Pointer 3)}: This section allows users to export the custom search results in various formats such as CSV, Excel, PDF. Additionally, users can print or copy the data for further analysis or reporting.
  \item \textbf{Classic and Modern View Switch Button (Pointer 4)}: This button allows users to toggle between the classic and modern views, giving users flexibility in selecting the interface they prefer for viewing data.
  \item \textbf{Search on this Page (Pointer 5)}: The search bar enables users to quickly search for specific data or reports within the current page, improving navigation and ease of use.
\end{itemize}

\subsection{Modern View Description}
The modern view of the website offers a clean and interactive design for displaying traffic data. Key features of this view include:
\begin{itemize}
  \item \textbf{Traffic Data Display}: The traffic data is displayed in an easy-to-read format with graphs and tables, allowing users to view detailed traffic volume data for specific periods and operators.
  \item \textbf{Data Customization and Filtering}: Users can apply custom filters to display data specific to their needs, such as selecting particular operators, network categories, or date ranges.
  \item \textbf{Export and Print Options}: The modern view allows users to export the data into various formats (CSV, Excel, PDF) or print the data for further offline analysis.
  \item \textbf{Live Connection Status}: The live connection status (e.g., "Traffic API connected") is shown at the top to indicate the real-time state of the data connection.
\end{itemize}
\textbf{Note:} The modern view offers enhanced functionality and data visualization, making it easier for users to interact with traffic data.
